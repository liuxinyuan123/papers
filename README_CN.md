### 队列提取 • 可重复流程 • 方法透明开放

本仓库存放一系列基于真实医学数据库（如 **MIMIC-IV**）的研究论文所使用的 **SQL 队列提取代码**。

每篇论文均包含独立文件夹，内含：

* 📊 队列提取 SQL
* 🧾 变量定义或数据字典（如适用）
* 📄 队列构建说明文档

（本仓库不公开任何分析代码、图表或患者级数据。）

---

## 🌟 仓库目的

* 📖 推动真实世界数据研究的透明化  
* 🧪 提供可重复的队列提取流程  
* 🧰 提供统一的 SQL 提取模板  
* 🔍 明确展示纳入 / 排除标准与变量逻辑  

---

## 🔒 数据隐私说明

本仓库 **不包含任何患者级数据**。  
仅提供 SQL 脚本与示例文档。  
使用 MIMIC-IV 等数据库需获得官方授权。

---

## 📝 使用方法

1. 获取目标数据库访问权限  
2. 克隆本仓库  
3. 进入对应论文文件夹  
4. 根据 文件夹内的README.md 文件，按照步骤执行提取数据。
